Advanced query search

  • You can use django field lookups to perform advanced searches
  • Some examples:
  • Users who have not changed their password since last admin’s request (on March 1, 2023): 
    extra_details__password_date_changed__date__lt:2023-03-01
  • Same as above with users’ account creation prior to admin’s request: 
    extra_details__password_date_changed__date__lt:2023-03-01
    AND date_joined__date__lt:2023-03-01
  • Same as above, but more accurate version with time: 
    extra_details__password_date_changed__lt:2023-03-01T12:00:00
    AND date_joined__date__lt:2023-03-01:2023-03-01T12:00:00
